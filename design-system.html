Dual Range Slider Component

Двойной ползунок для выбора диапазона (возраст, цена). Два <input type=range> наложены на один трек. JS-функция updateAgeSlider() обновляет заливку трека через left и width.

Live Preview — Age Range
22 45
1860
<div class="range-track">
  <div class="range-track-fill" id="age-fill"></div>
  <input type="range" id="age-min" min="18" max="60" value="18"
    oninput="updateAgeSlider()">
  <input type="range" id="age-max" min="18" max="60" value="60"
    oninput="updateAgeSlider()">
</div>

function updateAgeSlider() {
  let mn = +ageMin.value, mx = +ageMax.value;
  if (mn > mx - 1) { ageMin.value = mx - 1; mn = mx - 1; }
  const pct = v => (v - 18) / (60 - 18) * 100;
  ageFill.style.left  = pct(mn) + '%';
  ageFill.style.width = (pct(mx) - pct(mn)) + '%';
}

Прогресс-бар цели с токенами. Создатель задаёт название, сумму и время — фаны скидываются чаевыми. При достижении цели автоматически запускается событие (live, контент, etc). Используется в карточках search.html, постах feed.html, профиле.

/* Card micro-variant */
<div class="card-goal">
  <div class="card-goal-label">${label}<span>${cur}/${max}🔥</span></div>
  <div class="card-goal-track">
    <div class="card-goal-fill" style="width:${pct}%"></div>
  </div>
  <div class="card-goal-sub"><span>${pct}% собрано</span><span>${left} осталось</span></div>
</div>

/* JS data shape */
goal: { label: 'Ночное Live-шоу 🌙', cur: 720, max: 1000 }

Flash Event Pattern

Tip Menu Pattern

Фиксированный прайс-лист действий в профиле создателя. Каждый пункт — кликабельная строка, автоматически инициирующая tip на нужную сумму. Снижает трение при отправке чаевых.

Auto-Responder Pattern

Чат-бот с настраиваемыми правилами: триггер → текст → задержка. Поддерживает переменные {имя}, {сумма}, {план}. 9 типов триггеров. Настраивается в model-cabinet → Автоответы.

Trigger Types
${['👋 Новый подписчик','💰 Чаевой','💎 Чаевой $50+','🔄 Продление','🔑 Ключевое слово','🔒 Покупка PPV','😢 Отписка','🕐 Неактивен 7д','🎂 День рождения'].map(t=>`${t}`).join('')} 
/* Rule structure */
{
  trigger: 'new_sub' | 'tip' | 'tip_big' | 'renew' | 'keyword' | 'ppv' | 'unsub' | 'inactivity' | 'birthday',
  text: 'Привет, {имя}! 👋 Добро пожаловать!',
  delay_min: 0,        // delay before sending
  keywords: ['цена'],  // only for 'keyword' trigger
  enabled: true
}

/* Available variables */
{имя} {сумма} {план} {месяцев} {дата} {ранг} {ссылка_профиля}
