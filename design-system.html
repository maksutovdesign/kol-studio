Accessibility WCAG 2.1 AA

✅ Реализовано
• Skip-to-content link
• :focus-visible outline 2.5px
• aria-label на всех иконках
• role=dialog на модалках
• aria-live для тостов
• Breadcrumb nav aria-label
• loading="lazy" на img
• lang="ru"/"en" на html
• Print CSS скрывает UI
• Keyboard shortcuts
⚠️ Будущие улучшения
• aria-expanded на FAB/nav
• aria-current="page" в nav
• Trap focus в модалках
• Escape key → close modal
• Reduced motion support
• Alt text на все изображения
• ARIA роли для таблиц
• Контраст 4.5:1 проверка

• ARIA live regions
